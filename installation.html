MetaboStatR: Installation Guide

This guide will walk you through the steps to install the MetaboStatR package and its dependencies.

System Requirements

  • R version 4.4.0 or higher.
  • RStudio Desktop (recommended).

Installation Steps

  1. Install R and RStudio

    If you don't have R and RStudio installed, you can download them from the following links:

    • Download R
    • Download RStudio Desktop

  2. Install the `devtools` package

    The devtools package is required to install MetaboStatR from its source. Open R or RStudio and run the following command in the console:

    install.packages("devtools")

  3. Install MetaboStatR

    Since you have the source code for MetaboStatR, you can install it using the devtools package. Make sure your R session's working directory is set to the folder containing the MetaboStatR source code (the folder with the `DESCRIPTION` file).

    Run the following command in the R console:

    devtools::install()

    Alternatively, if this was on GitHub, you would use a command like this, replacing `user/repository` with the actual path:

    # devtools::install_github("user/repository")

  4. Install Required Dependencies

    MetaboStatR depends on several other R packages from both CRAN and Bioconductor. The following R script will install all the necessary packages.

    You can run this script in your R console:

    
# CRAN Packages
cran_packages <- c(
    "caret", "digest", "dplyr", "ggplot2", "glmnet", "gridExtra", "Hmisc",
    "htmlwidgets", "magrittr", "mixOmics", "patchwork", "pheatmap", "plotly",
    "pROC", "purrr", "RColorBrewer", "readxl", "stringr", "svglite",
    "tibble", "tidyr", "webshot2", "ggstatsplot", "grid", "knitr", "openxlsx",
    "rmarkdown", "rstudioapi", "testthat", "tools"
)

# Bioconductor Packages
bioc_packages <- c("pmp", "ropls", "vsn", "BiocManager")

# Install CRAN packages if not already installed
for (pkg in cran_packages) {
    if (!requireNamespace(pkg, quietly = TRUE)) {
        install.packages(pkg)
    }
}

# Install Bioconductor packages if not already installed
for (pkg in bioc_packages) {
    if (!requireNamespace(pkg, quietly = TRUE)) {
        if (!requireNamespace("BiocManager", quietly = TRUE)) {
            install.packages("BiocManager")
        }
        BiocManager::install(pkg)
    }
}

cat("\nAll dependencies have been checked and installed if missing.\n")

Once all these steps are complete, you are ready to use the MetaboStatR package!
